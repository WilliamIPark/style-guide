STYLE GUIDE

1. Logo Usage

The monogram is an extremely important part of the brand. It represents not only the brand,but me personally therefore care should be taken in every way to make sure it is recognisable in every state. Care has been put into making the logo proportional on a grid so it can be recreated and changed without destroying the essense of the shape.

Large Version

The large version of the logo should only be used in areas where the shape is well defined. For example, on the main logo image on a banner of a website.

Small Version

The small version should be used in areas where definition of the logo is lost, especially if the gap becomes hard to see. For example, a favicon.

Logo Usage / Other Usage Restrictions

Do not do any of the following:

  • Colour the logo in a colour that is not listed in the brand's colour palette.
  • Use multiple colours for the logo.
  • Rotate the logo.
  • Resize the logo out of proportion.

2. Brand Colours

A consistant colour pallette is something important to the brand. The dark gray and red contrast really well and it stands its ground as a pillar in the brand's design. The main gray (#4F4F4F) and main red (#F22267) are supported by other colours in the same tonal range, which should only be used in moderation. An example for using the other colours include using the lighter grays for toning back some text that is otherwise to strong in the hierarchy. Another example would be to put them in use for interactive elements, such as to create a hover effect for a button.

3. Typography

Josef Müller-Brockmann lived his life in design, and was a pioneer in objective design - he believed that if design was objective that it would be timeless. This is why I chose Futura as my typeface of choice to support this brand. It is geometric, objective, and in my eyes, timeless. It is easy to read at small sizes and is beautiful at large sizes, and comes with a lot of weights to allow typographic freedom in a single typeface.

Typography / Wordmark Usage

WILLIAMPARK

The top of the monogram must be in line with the top of the wordmark, and the same must apply to the bottom. If the monogram is used in a different location than shown it must be either vertically or horizontally proportional to the wordmark. If the wordmark does not work in either of these cases, don't use it.

Typography / Type Styles

I use “Minor Third” on type-scale to get the size of my headers and text. Try to follow the font style for the headers and also the em in relation to other headers, but feel free to modify the base size of the headings to increase or decrease the size of them all together. I use 18px/1.125em for a base size which you should also use as a starting point if designing for screen and adjust from there if needed.
